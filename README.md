# 💻 Algoritmos e Programação em C

Coleção de projetos e exercícios desenvolvidos para aprofundar o conhecimento em linguagens de baixo nível e lógica de programação.

## 🛠️ Projetos em Destaque

### 🧮 Calculadora Multiplataforma Híbrida (./Calculadora-Multiplataforma)
Uma calculadora funcional desenvolvida para rodar perfeitamente em múltiplos ambientes.
- **Diferencial Técnico:** Uso de diretivas de pré-processador (`#ifdef _WIN32`) para identificar o Sistema Operacional e executar comandos de limpeza de tela específicos (`cls` ou `clear`).
- **Recursos implementados:**
  - Operações básicas com validação de divisão por zero.
  - Estrutura de repetição `do-while` e controle de fluxo com `switch`.
  - Manipulação de buffer de teclado para entrada de dados limpa.

 ### 📋 **Manipulação de Arrays e Lógica** (./Exercícios-Arrays)
* Exercícios de percorrimento de vetores e matrizes para cálculos estatísticos e filtragem de dados.
